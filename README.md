# Border Fixed

##Include:

```html
    <link href="/css/border-fixed.css" rel="stylesheet">
```

## Using this `border-fixed.css`:

```html
<div class='border-fixed'>
    <div class='top'></div>
    <div class='right'></div>
    <div class='bottom'></div>
    <div class='left'></div>
</div>
```
