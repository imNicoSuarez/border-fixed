# Frame it

##Include:

```html
    <link href="/css/frame-it.css" rel="stylesheet">
```

## Using this `frame-it.css`:

```html
<div class='frame-it'>
    <div class='top'></div>
    <div class='right'></div>
    <div class='bottom'></div>
    <div class='left'></div>
</div>
```
